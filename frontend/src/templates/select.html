Классификация

Метод классификации используется для определения категории, к которой принадлежит новый объект на основе обучающей выборки.

Кластерный анализ

Кластерный анализ позволяет группировать объекты на основе их схожести без предварительных меток.

Регрессионный анализ

Регрессионный анализ используется для предсказания вещественных значений: стоимости недвижимости, акций и тп.
