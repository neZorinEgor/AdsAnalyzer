Перед тобой данные по трём кластерам рекламных кампаний. Проанализируй их по следующей структуре:

    Общая характеристика кластеров:

        Опиши размер каждого кластера (количество кампаний)

        Отметь общий уровень эффективности (высокий/средний/низкий)

    Ключевые различия между кластерами:

        Сравни основные метрики (CTR, wCTR, показы, клики, прибыль)

        Учитывай как средние значения, так и распределение (min, max, std)

        Особое внимание удели SHAP-значениям, которые показывают важность различий

    Портрет каждого кластера:
    Для каждого кластера (0, 1, 2) опиши:

        Основные характеристики (какие метрики выделяют его)

        Сильные и слабые стороны

        Особенности поведения пользователей (на основе глубины просмотра, отказов)

        Финансовую эффективность (прибыль, расходы)

        Интерпретацию PCA-компонент (если возможно)

    Рекомендации:

        Для какого кластера реклама наиболее эффективна? Почему?

        Какие проблемы видны в каждом кластере?

        На какие метрики стоит обратить внимание для улучшения?

    Выводы:

        В чем принципиальные различия между кластерами?

        Какие кластеры требуют оптимизации?

        Есть ли аномалии в данных?

Анализируй данные последовательно, обращая внимание на:

    Абсолютные значения метрик

    Относительные различия между кластерами

    SHAP-значения, показывающие значимость различий

    Соотношение показателей внутри каждого кластера
